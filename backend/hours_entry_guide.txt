FOOD LOCATION HOURS ENTRY GUIDE
============================================================

HOW TO FIND HOURS:
1. Search Google for: [location name] + [address] + hours
2. Check the location's website (if listed)
3. Call the phone number (if listed)

TIME FORMAT:
- Use 24-hour format: 09:00, 14:30, 17:00
- Or 12-hour with AM/PM: 9:00 AM, 2:30 PM, 5:00 PM

EXAMPLES:
monday_open: 09:00
monday_close: 17:00
monday_notes: By appointment only

SPECIAL CASES:
- If closed all day: leave open/close blank
- If 24/7: set is_24_7 to 'true'
- If same hours every day: copy the values
- If hours vary: add note in hours_notes column
